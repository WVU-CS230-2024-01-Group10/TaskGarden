<!-- GETTING STARTED -->

### Prerequisites

Setting up the project locally
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/WVU-CS230-2024-01-Group10/TaskGarden.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the app
   ```sh
   npm start
   ```
